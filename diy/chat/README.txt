#### Back-end

Há dois exemplos de server (server.php e anotherserver.php). O chat e
game das carinhas funciona com ambos, contanto que a URL seja a mesma
que está configurada nos arquivos dos servidor. Para que funcione com
outros dispositivos, é preciso alterar onde estiver "localhost" para
o ip da sua máquina. Verifique também suas configurações de firewall -
caso esteja ativado - para que a porta esteja liberada para escuta.

Qualquer dúvida só enviar email: contato@mariannactx.com
